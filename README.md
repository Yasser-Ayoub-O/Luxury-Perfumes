# Luxury Perfumes Website

## 📌 Overview
Luxury Perfumes is a responsive landing page designed for a perfume brand.  
The project was built using HTML and CSS with a traditional float-based layout to strengthen core CSS fundamentals before moving to modern layout systems like Flexbox and Grid.

---

## 🚀 Features
- Clean and elegant landing page design
- Float-based layout structure (no Flexbox/Grid)
- Responsive sections for better usability
- Product showcase sections
- Simple hover effects for better UI experience

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Float-based layout)

---

## 🎯 Purpose of the Project
This project was created as a practice exercise to:
- Master core CSS layout techniques using floats
- Improve understanding of traditional web layout systems
- Prepare for modern layout techniques like Flexbox and Grid
